Fonema service with Odoo

To use this module, you need an account with Monema.

You can create an account by visiting this page: https://www.monema.es/en/try-our-virtual-pbx-for-free/

When receiving a call you will be able to see the name of the contact, access their file if you have the Chrome extension installed, as well as keep a small record of the contact's incoming and outgoing calls by storing it in your chatter (contact history).

What are the advantages of Fonema integration with Odoo?

✓ Caller ID

By activating the integration with Odoo, every time you receive a call the caller's phone number will be searched among the Odoo contacts.

If you receive the call on your computer with softphone or physical IP phone, you will be able to see the name of the associated contact.

And on your mobile, with the Fonema app, you will receive a PUSH notification with the name of the contact.

✓ Call annotation

View the call log from the chatter (contact history) of the contact.

✓ Access to the contact file

If you have the Fonema Chrome extension installed, you can access the customer's file as soon as you pick up an incoming call from the contact and at the time you make a call to him.
